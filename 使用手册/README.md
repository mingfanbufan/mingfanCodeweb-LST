#### 4.2 大学学习资料
按「课程-实验-报告」三级结构组织，样式如下：
```
大学学习资料/
├── Linux操作系统/
│   ├── 实验1：openEuler安装（图文）
│   ├── 实验2：LVM卷扩容（含视频）
│   └── 实验报告模板（.docx）
├── 数据库技术及应用/
│   ├── MySQL8.0离线安装包（本站镜像）
│   ├── 实验：主从复制+读写分离
│   └── 期末复习思维导图（.xmind）
├── 云计算/
│   ├── FusionCompute6.5.1镜像（.iso）
│   ├── 桌面云实验手册（.pdf）
│   └── OpenStack搭建脚本（shell）
└── …
```

**用法**  
① 左侧目录树可折叠，点击直接跳转锚点；  
② 所有 `*.iso`、`*.rpm`、`*.deb` 均托管在「下载站」，提供阿里云盘备用链；  
③ 实验报告支持「一键复制到 Typora」按钮，自动去掉广告页眉。

### 九、常见问题（FAQ）

**Q1 下载站链接失效？**  
A 先在 GitHub Issues 搜索「文件名」，确认是否已报备；若无，请新建 Issue，站长 24h 内补链。

**Q2 是否可以批量爬取？**  
A 允许善意爬取，但请遵守 `robots.txt` 限速（≤ 60 次/分钟）；如要全站镜像，请使用官方每日凌晨打包的 `static.tar.gz`，避免消耗服务器流量。

**Q3 实验环境与生产环境冲突？**  
A 所有脚本均在 `/scripts` 目录提供 `install.sh` 与 `uninstall.sh`，支持一键回滚；建议全程在虚拟机或 Docker 容器内操作。

**Q4 如何获得更新推送？**  
A GitHub 仓库 Release 页面订阅即可，每次大版本会更改日志；或关注首页「🔔 RSS」图标。

**Q5 能否捐赠？**  
A 站点维护费用目前由广告位（仅 1 个）+ 站长自费覆盖；如需请喝咖啡，首页底部「赞助」有微信/支付宝二维码，**完全自愿**。
