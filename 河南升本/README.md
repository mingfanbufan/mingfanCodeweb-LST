## [高等数学](高等数学/README.md)

专升本英语是专升本考试中所有考生都需要参加的考试科目。
它是对专科阶段英语学习成果的检验，主要考查学生在英语语言知识、语言运用能力、阅读理解能力、写作能力等方面是否达到本科阶段学习的基本要求。
专升本英语考试的目的是选拔具有较好英语水平的学生进入本科阶段学习，为学生在本科阶段学习专业课程中的英语文献阅读、国际交流等活动奠定基础。
